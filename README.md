# AstonCV - CV Database Website

This project is a dynamic CV database website developed using PHP and MySQL as part of the DG1IAD Internet Applications and Databases module at Aston University.

## Features
- User registration and login system
- Users can create and update their CV
- Public users can view all CVs
- Search CVs by name or programming language
- Secure password hashing
- SQL injection protection using prepared statements
- CSRF protection
- Session-based authentication
- Input validation and output escaping

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## How to Run
1. Import `cvs.sql` into MySQL/phpMyAdmin
2. Update database credentials in `config.php`
3. Run the project on a local server (XAMPP) or host server
4. Open `index.php` in a browser
